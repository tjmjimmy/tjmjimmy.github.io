超链接设置

使用[解释说明](链接地址)，注意是英文字符
[www.tjmjimmy.top](www.tjmjimmy.top)
www.tjmjimmy.top

快捷模式：下面格式相当于申明了[www.tjmjimmy.top]自带了链接
[www.tjmjimmy.top]:www.tjmjimmy.top
在后续的使用格式：
* + 空格[www.tjmjimmy.top]  这样[]内的文字也有了超链接
  • www.tjmjimmy.top
  • www.tjmjimmy.top
  • www.tjmjimmy.top
  • www.tjmjimmy.top
